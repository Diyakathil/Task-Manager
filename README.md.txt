Task Manager / To-Do App
A simple and interactive To-Do List app created using Python. It helps you manage your daily tasks, track completed ones, and save/load tasks from a file.

You can run it as:

>  A CLI (Command-Line Interface) app, or

> A GUI (Graphical User Interface) using Tkinter

Features

> Add new tasks

> View all tasks

> Mark tasks as complete

> Delete completed tasks

> Save tasks to a file

> Load tasks from a file

> GUI version with buttons and text boxes (Tkinter)

Technologies Used

> Python 3.x

> Object-Oriented Programming (OOP)

> Tkinter (for GUI)

> File Handling (for saving and loading tasks)

> Command-Line Interface (optional)
